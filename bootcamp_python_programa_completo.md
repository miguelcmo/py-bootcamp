# BOOTCAMP DE PYTHON - PROGRAMA COMPLETO
## De Fundamentos a Aplicaciones Web

### INFORMACIÓN GENERAL
- **Duración total:** 60 horas
- **Sesiones virtuales:** 8 sesiones (32 horas) - Teoría y conceptos
- **Sesiones presenciales de proyecto:** 6 sesiones (24 horas) - Desarrollo guiado
- **Sesión final de presentación:** 1 sesión (4 horas) - Demo day

---

## FASE 1: FUNDAMENTOS DE PYTHON (Sesiones 1-4)

### SESIÓN 1 - VIRTUAL (4 horas): Introducción a Python y Primeros Pasos

**Objetivos:**
- Configurar el entorno de desarrollo
- Comprender la sintaxis básica de Python
- Trabajar con variables y tipos de datos
- Realizar operaciones básicas

**Contenido:**
1. **Introducción (30 min)**
   - ¿Qué es Python y por qué aprenderlo?
   - Instalación de Python y configuración del entorno (VS Code, Python 3.x)
   - Introducción a la terminal/consola
   - Primer programa: "Hello World"

2. **Variables y Tipos de Datos (90 min)**
   - Variables: declaración y nomenclatura
   - Tipos de datos básicos:
     - Strings (cadenas de texto)
     - Integers (enteros)
     - Floats (decimales)
     - Booleans (booleanos)
   - Conversión entre tipos (casting)
   - Operadores aritméticos, de comparación y lógicos

3. **Strings en Profundidad (60 min)**
   - Métodos de strings
   - Concatenación y formateo (f-strings, .format())
   - Slicing y manipulación
   - Ejercicios prácticos

4. **Input del Usuario y Debugging Básico (30 min)**
   - Función input()
   - Comentarios en el código
   - Errores comunes y cómo solucionarlos
   - Print debugging

**Ejercicios prácticos:**
- Calculadora simple
- Programa de saludo personalizado
- Conversor de unidades

**Tarea:**
- Crear un programa que solicite datos al usuario y genere un perfil personalizado

---

### SESIÓN 2 - VIRTUAL (4 horas): Estructuras de Control

**Objetivos:**
- Implementar estructuras condicionales
- Usar bucles para repetir código
- Combinar estructuras de control

**Contenido:**
1. **Condicionales (90 min)**
   - Estructura if
   - if-else
   - if-elif-else
   - Operadores lógicos (and, or, not)
   - Condicionales anidados
   - Expresiones condicionales (ternario)

2. **Listas (60 min)**
   - Creación y acceso a listas
   - Métodos de listas (append, insert, remove, pop)
   - Slicing de listas
   - Listas por comprensión (introducción)

3. **Bucles (90 min)**
   - Bucle for
   - Bucle while
   - Break y continue
   - Range y enumerate
   - Bucles anidados
   - Iteración sobre listas

**Ejercicios prácticos:**
- Sistema de calificaciones
- Juego de adivinanza de números
- Procesador de lista de compras

**Tarea:**
- Crear un menú interactivo para una aplicación simple

### SESIÓN 3 - VIRTUAL (4 horas): Estructuras de Datos

**Objetivos:**
- Dominar diccionarios y tuplas
- Trabajar con estructuras de datos complejas
- Comprender cuándo usar cada estructura

**Contenido:**
1. **Diccionarios (90 min)**
   - Creación y sintaxis de diccionarios
   - Acceso y modificación de valores
   - Métodos de diccionarios (keys, values, items, get)
   - Diccionarios anidados
   - Iteración sobre diccionarios
   - Dictionary comprehension

2. **Tuplas (45 min)**
   - Características de las tuplas
   - Cuándo usar tuplas vs listas
   - Unpacking de tuplas
   - Operaciones con tuplas

3. **Sets (45 min)**
   - Conjuntos y sus propiedades
   - Operaciones de conjuntos
   - Cuándo usar sets

4. **Estructuras Combinadas (60 min)**
   - Listas de diccionarios
   - Diccionarios con listas como valores
   - Ejercicios de estructuras complejas

**Ejercicios prácticos:**
- Sistema de registro de usuarios
- Inventario de productos
- Agenda de contactos

**Tarea:**
- Crear una base de datos simple usando estructuras de datos

### SESIÓN 4 - VIRTUAL (4 horas): Funciones y Modularización

**Objetivos:**
- Crear y usar funciones
- Entender el scope de variables
- Importar y usar módulos

**Contenido:**
1. **Funciones Básicas (90 min)**
   - Definición de funciones
   - Parámetros y argumentos

   - Argumentos por defecto
   - *args y **kwargs
   - Docstrings

2. **Scope y Variables (45 min)**

3. **Módulos y Paquetes (90 min)**
   - Importación de módulos
   - Módulos de la biblioteca estándar (math, random, datetime)
   - Crear propios módulos
   - Estructura de proyectos Python
   - Virtual environments (venv)

4. **Lambda y Funciones de Orden Superior (45 min)**
   - Funciones lambda
   - Map, filter, reduce
   - Funciones como argumentos

**Ejercicios prácticos:**
- Calculadora con funciones

- Sistema de utilidades personalizadas

- Crear un módulo de utilidades reutilizable

## NOTAS FINALES

Este bootcamp está diseñado para llevar a los estudiantes desde cero hasta la construcción de una aplicación web funcional en Python. La progresión es cuidadosa y cada sesión construye sobre la anterior.

**Recomendaciones para el instructor:**
1. Enfatizar la práctica sobre la teoría
2. Usar ejemplos del mundo real
3. Fomentar el trabajo en equipo
4. Realizar code reviews regularmente
5. Estar disponible entre sesiones para consultas
6. Adaptar el ritmo según el progreso del grupo

**Para los estudiantes:**
- Dedicar tiempo entre sesiones a practicar
- No tener miedo de experimentar y cometer errores
- Hacer preguntas cuando algo no esté claro
- Trabajar en equipo y compartir conocimientos
- Documentar el aprendizaje (blog, notas, etc.)

¡Éxito con el bootcamp!
